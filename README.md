# hotel_reservation_service_management_system
This project is a complete Hotel Reservation and Service Management System built to handle room bookings, customer services, and secure online payments. It delivers a clean, responsive user interface and smooth user experience for both guests and administrators. The system is suitable for real-world hotel operations, portfolio projects, or academic assignments.

Core Features

Reservation System

Browse available rooms with images, features, and pricing

Real-time room availability check

Guest booking form with date selection

Email Booking confirmation page

Email/in-page booking summary

 Payment Gateway

Integrated online payment system

Supports deposit or full-payment options

Secure processing and clear payment status

Booking stored only after successful payment

 Service Management

Hotel services displayed (e.g., cleaning, food delivery, spa)
